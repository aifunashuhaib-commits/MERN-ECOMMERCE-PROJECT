# MERN E-commerce Basic ()
This archive contains a minimal MERN stack e-commerce project with:
- backend/ (Express + Mongoose)
- frontend/ (Vite + React)

Follow individual READMEs in backend/ and frontend/ to run.

Notes:
- Create an admin using POST /api/admin/register (one-time) or seed using MongoDB GUI.
- After login POST /api/admin/login you'll receive a token for protected admin routes.
